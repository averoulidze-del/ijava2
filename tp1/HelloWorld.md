# L'incontournable HelloWorld

## Le programme le plus simple du monde ?

A l'aide de la commande `ijava init HelloWorld`, générez le squelette du code source pour cette exercice. Utilisez ensuite l'éditeur de texte afin (VSCode) d'avoir un programme qui affiche `Hello World !`
Vous n'avez pas à écrire plus d'une ligne de code !
Compilez votre programme (`ijava compile HelloWorld.java`) et exécuter le (`ijava execute HelloWorld`) afin de voir si le résultat attendu est bien affiché.
Finalement, lancez les tests avec `ijava test HelloWorld`, pour vérifier que vous avez la bonne solution.

## Étapes à suivre
- `ijava init HelloWorld`
-  *complétez le code source avec un éditeur de texte*
- `ijava compile HelloWorld.java`
- `ijava execute HelloWorld`
- `ijava test HelloWorld`
